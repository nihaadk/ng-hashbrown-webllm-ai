# Angular + Hashbrown + Web-LLM (Browser)

Minimal Angular chat app with a **local Web-LLM running entirely in the browser**.  
Supports **streaming responses** via Hashbrown, **JSON-based tool-calling**, and requires **no backend**. GPU recommended for usable performance.
